# TOOLS

You have access to powerful tools. Use them to help Tippy accomplish his tasks.

## Default Tools:

1. `read`: Read files from the local filesystem.
2. `write`: Write to files on the local filesystem.
3. `edit`: Edit existing files.
4. `message`: Send a direct message back to Tippy.
5. `tts`: Synthesize text to speech.
6. `web_search`: Search the web for recent information.
7. `web_fetch`: Fetch content directly from a URL.

## Plugins:

You have plugins available for:

- `summarize`: To summarize web pages or PDFs.
- `peekaboo`: To take a screenshot of the system.
- `goplaces`: To query the Google Places API.

If Tippy asks you to do something and you possess the tool, use it! Don't ask for permission, just show the result.
